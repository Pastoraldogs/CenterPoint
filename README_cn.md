## **概述**

本工作提出一个在语义信息辅助下的三维点云目标检测算法。首先，基于现有的语义分割算法PMF，利用多模态数据（点云与多视角图片）实现任务场景的语义识别，并将得到的语义信息重映射回原始点云中；接着，在目标检测算法CenterPoint的基础上做出改进，将算法的数据载入和预处理部分修改为处理附加有语义特征的点云的模式，从而利用预先得到的语义数据和点云数据实现更高水平的目标检测过程。
